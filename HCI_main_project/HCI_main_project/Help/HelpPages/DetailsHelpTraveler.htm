Help - Tour Details Tripago

Tour details page represents details about specific tour. It can be opened from Home page or from History page. Every user can see description of the tour and all side activities. In the bottom part of the page user can see lists of all restaurants, accommodations and attractions with their pins on map. Besides offering tour details this page is used for tour manipulation, including reserving, booking or canceling tour.

Need help for a specific page part? Click to jump to specific help part:

  • Tour reservation
  • Tour booking
  • Tour canceling

● Help - Tour reservation

You can reserve opened tour by clicking "Reserve" button, which will open a new dialog where you need to confirm reservation by clicking "Confirm reservation" button. After reservation there will be new button for canceling reservation (Tour canceling).

● Help - Tour booking

You can book new tour or reserved one by clicking "Book now" button, which will open a new dialog where you need to confirm booking by clickin "Confirm booking" button. After booking there will be new button for canceling reservation/booking (Tour canceling).

● Help - Tour canceling

If you already have reserved or booked tour there should be "Cancel reservation" button. You can cancel you reservation by clicking that button, which will open a new dialog where you need to confirm you action. After canceling the reservation you can make new reservation or book a tour (Tour reservation and Tour booking).
