Help - Homepage Tripago

Homepage is the main page of the application.
On the left there's the navbar which enables you to switch different views on the right, and choose if you want to see available Tours, Attractions, Accommodation, Restaurants, Tour statistics, or open Help window (like this one).
On the right is the card pane, with scrolling view of different types of cards on the bottom, and search & filter bar on the top.

● Help - Navbar

Navbar is the main source of navigation in the app. It allows you to choose from 4 different card display types:

  • Tours
  • Accommodation
  • Attractions
  • Restaurants

Selecting either one of these options will change the content and look (slightly) of the cards on the right, displaying information for the chosen entities.
Read more about different card displays here.

Right below those four type-oriented options, there's an option for you to open "Tour Statistics".
This option will open up the statistics view on the right pane, showing total income, total reservations and total bookings, alongside stats for each of the individual tours.
More on that here.

"Help" navbar option will showcase the same behaviou as pressing "F1" key on your keyboard will - opening the Help broswer window, with documentation and explanation of the page in focus.

Tour Card

Tour Card is designated for displaying tours. It shows tour image, price, name, dates => basic information about each tour.

The bottom row of the card holds action button:

  • - delete tour:
  • - edit tour
  • Information - open tour information details

Tour Card - Information

Disclaimer: Information applies for both card types (Tour and Simple card).

Clicking on the "Information" button will open a window with detailed information about the tour. Besides the information on the card, tour description and tour attractions, accommodation and restaurant will show.
Handy Map view of all information will also be displayed.

To read about tour information page, click "F1" or "?" button on the information page of your interest (while running the app).

Delete a Card

Disclaimer: Information applies for both card types (Tour and Simple card).

To delete a card (aka the item associated with the card), simply click on the button. Clicking on the button will open up the "Are you sure you want to delete ____" window, so you don't have to worry - even if you accidently click the delete button or click the delete button of the wrong item, the window will show you the item selected for deletion and ask you if that is want you want to do.
From there just choose "Delete ___" button to confirm deletion, or "Cancel" to close the window and go back, no data lost.

CAUTION: Deleting is a permanent action, you won't be able to undo it once you confirm deletion.

● Help - Tour Statistics

Tour Statistics navbar button will open the overview of all Tours you are managing through the app.

It gives you three main calculations, righ below search & filter section:

  • Total income - sum of prices of all reservation and bookings you sold
  • Total reservations - total number of reservation across all tours
  • Total bookings - total number of bookings across all tours

It also displays a elongated version of Tour cards, with all the same infomation as before (in a bit of a different layout), as well as the number of reservations and bookings for that specific tour.

You can also click on any of the cards' "Information" button to see tour details from here as well, same bahviour as mentioned before.

● Help - Search

Search bar is located in the top right corner of the Homepage.

By typing text into the search bar, the cards displayed will change to include only the items with the name that contains the text typed in the search bar.

No need to click any buttons, just type and the list of items will filter with each written letter.

● Help - Sort

Sort drop down menu is located to the right of the Search bar. Sorting is only available for tours (both in the "Tours" and "Tour Statistics" view).

There are five sorting criteria:

  • Most popular - cards will be sorted by the number of reservations and bookings (in sum) so that the first tour shown is the "most popular"
  • Price lowest - cards will be sorted by the price so that the first tour shown is the the cheapest
  • Price highest - cards will be sorted by the price so that the first tour shown is the the priciest
  • Date desc - cards will be sorted by the the "from" date, so that the first tour shown starts the latest
  • Date asc - cards will be sorted by the the "from" date, so that the first tour shown starts the soonest

Default sorting option is "Most popular".

● Help - Filter

Filter button is located in the top right corner of the Homepage.

When you first open the app, the filters are hidden. To make them visible simply click on the filter button => filter options outlined with gray border container will appear:

Now you can choose to filter the items shown by different criteria, based on the item type:

  • Tours - filter by min and max price, as well as date range
  • Attractions - filter by attraction location (choose city option from the drop down menu)
  • Accommodation - filter by accommodation location (choose city option from the drop down menu), as well as accommodation type (choose type option from drom down menu)

When you're happy with the choices for filters you inputed, click on the "Apply filters" button to filter the card selection - only the cards who's content matches the filter inputs will remain.

Filtering takes into consideration the search and sort values (if provided) so the final selection will contain cards that match all three criteria (given by search, sort and filters).

To clear all filters at the same time, press the "Clear filters" button - cards will update to include only the criteria left by search and sort (if they exist).
