Help - Tour Details Tripago

Tour details page represents details about specific tour. It can be opened from Home page or from History page. Every user can see description of the tour and all side activities. In the bottom part of the page user can see lists of all restaurants, accommodations and attractions with their pins on map. Besides offering tour details this page is used for displaying all reservations to the agent.
